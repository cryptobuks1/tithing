HOW TO INSTALL 


SYSTEM REQUIREMENT.
1. DOWNLOAD XAMPP AND INSTALL. DEPENDING ON THE OPERATION SYSTEM PREFERENCE
2. MOVE THE ROOT DIRECTORY INTO THE htdoc FOLDER IN THE XAMPP FOLDER

3. START THE SERVERS FOR BOTH PHP AND MySQL

4. NAVIGATE TO http://localhost IN ANY BROWSER

5. NAVIGATE TO phpMyAdmin and CREATE A DATABASE 'gracebaptist'
 
6. NAVIGATE TO THE 'gracebaptist' DATABASE THAT WAS JUST CREATED AND IMPORT THE DATABASE FILE IN THE 'databaseFolder' 

AFTER SUCCESSFUL INSTALL 

7. YOU CAN NOW VISIT http://localhost/tithing IN CASE YOU DON'T CHANGE THE DIRECTORY NAME
